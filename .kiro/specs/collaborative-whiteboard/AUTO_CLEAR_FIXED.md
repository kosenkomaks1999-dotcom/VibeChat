# Автоочистка доски исправлена

## Проблема
Когда вы выходили из комнаты и заходили обратно, старые рисунки оставались на доске, даже если комната была пустая.

## Решение
Добавлена **автоматическая очистка доски** при входе в пустую комнату.

### Как работает:

1. **При создании комнаты:**
   - Проверяется количество пользователей
   - Если вы единственный (count === 1) - очищаются старые данные доски

2. **При входе в существующую комнату:**
   - Проверяется количество пользователей
   - Если вы единственный (count === 1) - очищаются старые данные доски
   - Также очищаются старые сообщения чата

3. **При выходе последнего пользователя:**
   - Слушатель отслеживает количество пользователей
   - Когда count становится 0 - доска очищается

## Логи

Теперь вы увидите в консоли:
```
[AUTO-CLEAR] Room was empty, clearing old whiteboard data
[AUTO-CLEAR] Old whiteboard data cleared successfully
```

## Тестирование

1. **Создайте комнату "test-auto-clear"**
2. **Откройте whiteboard и нарисуйте что-то**
3. **Закройте whiteboard**
4. **Выйдите из комнаты**
5. **Войдите обратно в "test-auto-clear"**
6. **Откройте whiteboard**
7. **Результат:** Доска должна быть ПУСТОЙ!

## Важно

- Данные очищаются только если комната была **полностью пустая**
- Если в комнате есть другие пользователи - данные сохраняются
- Это работает для всех комнат автоматически
